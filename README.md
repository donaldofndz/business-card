# Business Card

Minimal Next.js app using the App Router, TypeScript, Tailwind CSS, ESLint, and Jest.

## Installation

```bash
npm install
```

## Run the app

```bash
npm run dev
```

Open http://localhost:3000 to view the Hello World page.

## Run tests

```bash
npm run test
```

## Run linting

```bash
npm run lint
```

## Folder structure overview

```text
.
├── jest.config.js
├── jest.setup.ts
├── src
│   └── app
│       ├── page.test.tsx
│       └── page.tsx
└── ...
```

## Features and changes

- Hello World page at `src/app/page.tsx`.
- Jest + Testing Library setup in `jest.config.js` and `jest.setup.ts`.
- Basic page render test in `src/app/page.test.tsx`.
